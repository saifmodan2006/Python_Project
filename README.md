# Python Project Collection

This repository is a personal collection of small Python projects, utilities, and learning examples. The code is organized into folders that group related scripts (games, tools, utilities). Each script is intended to be run independently from the command line.

Contents

- `21_python_project/` — a set of simple games and utilities (number guessing, hangman, calculators, recorders, etc.).
- `python-project/` — another collection of small programs and practice scripts.

Requirements

- Python 3.8 or newer is recommended.
- Most scripts use only the Python standard library. Some scripts may require third-party packages; check the file header or top of the script for specific imports and install them with `pip` if needed.

Running examples

Run a script from the repository root. For example:

```
python "21_python_project/01_number_guessing.py"
```

Or run a script from the other folder:

```
python "python-project/01_calculator.py"
```

Files and structure

- Each file is a self-contained example or small app. Open the file in an editor to read usage notes and any dependency information at the top of the script.
